Gmail Integration
Gmail API Connection
Checking connection...
Setup steps:
1. Go to Google Cloud Console and create a project
2. Enable the Gmail API
3. Create OAuth 2.0 credentials (Web application)
4. Set redirect URI to your Vercel domain + /api/gmail?action=callback
5. Add GMAIL_CLIENT_ID, GMAIL_CLIENT_SECRET, GMAIL_REDIRECT_URI to Vercel env vars
6. Click "Connect Gmail" above, authorize, then save the refresh token as GMAIL_REFRESH_TOKEN
